      • About Course

      Belajarlah dengan cara terbaik untuk membuat sebuah website menggunakan HTML, CSS, Javascript, PHP, dan MySQL.

      HTML, CSS, Javascript, PHP, dan MySQL adalah lima pemrograman dasar untuk pembuatan website. Dengan demikian, siapapun yang tertarik untuk membuat sebuah website dari sisi coding pasti akan berjumpa dengan kelima bahasa atau script pemrograman tersebut.

      Video course ini merupakan yang terlengkap mengupas kelima pemrograman: HTML, CSS, Javascript, PHP, dan MySQL menggunakan Bahasa Indonesia di Udemy. Tidak peduli apakah Anda pernah memiliki website atau belum pernah melakukan pemrograman, video course ini jelas untuk Anda.

      Masing-masing bahasa atau script program tersebut memiliki fungsi spesifik, yaitu:

      • HTML: untuk merancang struktur halaman website. Seluruh website dibangun dari struktur HTML.
      • CSS: untuk mengatur bagaimana website akan terlihat di layar monitor. Indah tidaknya tampilan website diatur menggunakan CSS.
      • Javscript: untuk membuat website interaktif. Javascript adalah script pemrograman berbasis client-side sehingga sering digunakan untuk mengatur interaksi antara pengguna dan website dengan cepat.
      • PHP: untuk membuat website lebih powerful karena bisa menangani logika dan penyimpanan database. Hampir sebagian besar website di dunia ini dibentuk oleh PHP.
      • MySQL: untuk menyimpan data-data yang diinput oleh user ke dalam sistem database.

      Video course ini menjelaskan kelima bahasa atau script pemrograman tersebut sehingga Anda bisa memahami dunia desain website dari sisi coding secara cepat dan komprehensif. Isi dari pembahasan meliputi:

      • Pengenalan HTML
      • Pembuatan paragraf menggunakan HTML
      • Mengenal hyperlink dan cara pembuatannya
      • Meletakkan foto dan gambar di dalam website
      • Membuat tabel di dalam website
      • Mendesain form dan membuat input
      • Pengenalan CSS dan bagaimana menambahkan ke dalam HTML
      • Contoh-contoh selector dan pengaturan teks serta image
      • Pengenalan variabel di lingkungan Javascript
      • Operator perbandingan dan struktur IF
      • Perulangan-perulangan pada Javascript
      • Cara menulis kode program PHP dan pengenalan variabel serta tipe data
      • Fungsi echo dan print pada PHP
      • Menggunakan if dan perulangan
      • Pengenalan database dan MySQL
      • Dasar-dasar menambah data baru
      • Menampilkan data di dalam sebuah tabel

      Video course ini dilengkapi garansi Udemy dalam bentuk 30 hari uang kembali. Jadi silakan tonton dan pelajari cara terbaik mendesain website menggunakan HTML, CSS, Javascript, PHP, dan MySQL.
